# Radio

Radios are used when only one choice may be selected in a series of options.

## When to use

- When the user should see all available options.

## Overrides

<Overrides
  name="Radio"
  component={RadioExports}
  whitelisted={[
    'Root',
    'RadioMarkInner',
    'RadioMarkOuter',
    'Label',
    'Input',
    'RadioGroupRoot',
  ]}
  renderExample={props => (
    <StatefulRadioGroup
      overrides={{RadioGroupRoot: props.overrides.RadioGroupRoot}}
      name="choose one option"
      initialState={{value: '2'}}

        First
      </Radio>
      <Radio overrides={props.overrides} value="2">

      <Radio overrides={props.overrides} value="3">

      </Radio>

  )}
